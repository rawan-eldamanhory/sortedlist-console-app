# C# SortedList Implementation

This project demonstrates the usage of the `SortedList` collection in C# through a simple console application.

## Concepts Covered
- Generic `SortedList<TKey, TValue>` operations
  - Add, Remove, RemoveAt
  - Access elements via indexer
  - ContainsKey and ContainsValue
- Non-generic `SortedList` example
- Displaying elements
- Capacity, Count, IsReadOnly, and IsFixedSize

## Technologies
- C#
- Console Application
- Visual Studio

## Purpose
This project is intended for learning and practicing C# collection fundamentals, focusing on the SortedList data structure.
